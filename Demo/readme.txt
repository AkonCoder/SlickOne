SlickOne敏捷开发框架
最新版本V1.2.0
=================================
主要功能描述：
1. 用户登录身份验证，随机加密算法，密码加盐(Salt)单向验证实现；
2. SSO多站点统一登录验证，一次登录即可访问集成的多站点；
3. Mvc页面安全访问；
4. WebAPI的安全访问；
5. 集成流程设计器，表单设计器；
6. 基础数据（公司/角色/用户）支持多集团，多租户和SAAS平台系统要求；
